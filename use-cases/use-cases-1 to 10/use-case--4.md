# USE CASE: 4 Generate Top 10 Fastest-Growing Countries Report

## CHARACTERISTIC INFORMATION

Goal in Context
As a data analyst I want to generate a report showing the top 10 fastest-growing countries so that I can identify emerging high-density areas.

### Scope

Global population database.

### Level

Primary task.

### Preconditions

• System is running.
• Historical population data available.
• Analyst logged in if required.

### Success End Condition

Top 10 fastest-growing countries report displayed and exportable.

### Failed End Condition

No report produced.

### Primary Actor

Data Analyst

### Trigger

Analyst requests top 10 fastest-growing countries report.

## MAIN SUCCESS SCENARIO
1. Analyst navigates to “Population Growth Report”.
2. Analyst requests top 10 report.
3. System retrieves historical population data.
4. System calculates growth rates.
5. System identifies top 10 countries.
6. System displays report.

## EXTENSIONS

• Missing historical data: notify user.
• Cancel: return to main menu.

## SUB-VARIATIONS

None.

## SCHEDULE

Report displayed in table or chart, exportable.
