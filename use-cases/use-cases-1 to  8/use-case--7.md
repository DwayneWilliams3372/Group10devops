# USE CASE: 7 Generate Country Report with Key Information

## CHARACTERISTIC INFORMATION

### Goal in Context

As a researcher I want a country report including code, name, continent, region, population, and capital so that I can access key information for each country in one place.

### Scope

Global country database.

### Level

Primary task.

### Preconditions

• System is running.
• Country data available (code, name, continent, region, population, capital).
• Researcher logged in if required.

### Success End Condition

Country report displayed and exportable.

### Failed End Condition

No report produced.

### Primary Actor

Researcher

### Trigger
Researcher requests full country report.

## MAIN SUCCESS SCENARIO

1. Navigate to “Country Report”.
2. Request full report.
3. Retrieve country data.
4. Display data in table format with sorting/filtering options.

## EXTENSIONS

• Missing field data: show “N/A”.
• Cancel: return to main menu.

## SUB-VARIATIONS

None.

## SCHEDULE

Full country report displayed and exportable.
