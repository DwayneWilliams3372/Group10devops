# USE CASE: 6 View Urban and Rural Population by Continent

## CHARACTERISTIC INFORMATION

### Goal in Context
As a data analyst I want to view urban and rural population for each continent so that I can analyze global urbanization trends.

### Scope

Global population database with urban/rural breakdown.

### Level

Primary task.

### Preconditions

• System is running.
• Population data including urban/rural breakdown available.
• Analyst logged in if required.

### Success End Condition

Population data displayed, readable, and exportable.

### Failed End Condition

No data displayed.

### Primary Actor

Data Analyst

### Trigger

Analyst selects a continent.

## MAIN SUCCESS SCENARIO

1. Navigate to “Urban vs Rural Population”.
2. Select continent.
3. Retrieve total, urban, and rural population.
4. Display data in table or chart.

   ## EXTENSIONS

   • Missing data: display “data unavailable”.
   • Cancel: return to main menu.

   ## SUB-VARIATIONS

   None.

   ## SCHEDULE

   Urban and rural population data displayed and exportable.
