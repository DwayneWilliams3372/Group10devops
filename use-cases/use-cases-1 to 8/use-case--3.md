# USE CASE: 3 View Average Population Density for Each Country

## CHARACTERISTIC INFORMATION

### Goal in Context

As a researcher I want to view the average population density for each country so that I can analyze how densely populated different regions are.

### Scope

Global population and area database.

### Level

Primary task.

### Preconditions

• System is running.
• Population and area data available.
• Researcher logged in if required.

### Success End Condition

Population density data displayed, sortable, and exportable.

### Failed End Condition

No data displayed.

### Primary Actor

Researcher

### Trigger

Researcher requests population density data.

## MAIN SUCCESS SCENARIO

1. Researcher navigates to the “Population Density” section.
2. Apply optional filters (region/continent).
3. System retrieves population and area data.
4. System calculates population density.
5. System displays results in table or chart.

## EXTENSIONS

• Missing data: display “data unavailable”.
• Cancel: return to main menu.

## SUB-VARIATIONS

None.

## SCHEDULE

Population density data displayed, sortable, and exportable.
