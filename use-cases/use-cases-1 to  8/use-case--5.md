# USE CASE: 5 View Cities in a District by Population

## CHARACTERISTIC INFORMATION

### Goal in Context

As a researcher I want to view all cities in a district organized by population (largest to smallest) so that I can assess population distribution within local areas.

### Scope

District-level population database.

### Level

Primary task.

### Preconditions

• System is running.
• Population data for district cities available.
• Researcher logged in if required.

### Success End Condition

Sorted city list displayed and exportable.

### Failed End Condition

No data displayed.

### Primary Actor

Researcher

### Trigger

Researcher selects a district.

## MAIN SUCCESS SCENARIO

1. Navigate to “District Cities”.
2. Select district.
3. Retrieve and sort cities by population.
4. Display cities and populations.

## EXTENSIONS

• Invalid district: display error.
• No data: display “no data available”.
• Cancel: return to main menu.

## SUB-VARIATIONS

None.

## SCHEDULE

Sorted district city list displayed, exportable.
